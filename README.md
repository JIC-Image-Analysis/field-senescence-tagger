# Image tagging

## Key bindings

| Action         | Key   |
| -------------- | ----- |
| Open directory | ``o`` |
| Tag as "good"  | ``1`` |
| Tag as "bad"   | ``2`` |
| Next image     | ``RightArrow``, ``l`` |
| Previous image | ``LeftArrow``, ``h`` |
| Toggle help    | ``h`` |
| Save           | ``s`` |
| Quit           | ``Esc``, ``q`` |

## To install

* Install node.js and npm
* Run ``npm install``

## To run

* Edit main.js and change images_dir to point to a directory containing images.
* Run ``npm start``
