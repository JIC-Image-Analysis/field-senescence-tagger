# Field image tagging

## Key bindings

| Action         | Key   |
| -------------- | ----- |
| Open directory | ``o`` |
| Quit           | ``Esc``, ``q`` |

## To install

* Install node.js and npm
* Run ``npm install``

## To run

* Edit main.js and change images_dir to point to a directory containing images.
* Run ``npm start``
